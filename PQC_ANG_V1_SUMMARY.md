# PQC波束预测模型 (Version 1) 总结

## 项目概述

基于参数化量子电路(Parameterized Quantum Circuit)的波束RSRP预测模型，实现了量子-经典混合架构。

## 算法核心组件

### 1. 数据处理流程
- **输入**: 8个选定波束的RSRP值
- **输出**: 256个波束的完整RSRP预测
- **数据集**: 336,000个样本，每个样本256维特征
- **数据划分**: 训练集75%(252,000)，验证集15%(50,400)，测试集10%(33,600)

### 2. 量子电路设计
- **量子比特数**: 8个
- **编码方式**: 角度编码(RY门)
- **电路结构**: Strongly Entangling Layers，3层深度
- **测量方式**: Pauli-Z基下的期望值测量

### 3. 经典回归组件
- **输入维度**: 8维(量子测量输出)
- **隐藏层**: 64维ReLU激活
- **输出维度**: 256维(完整波束预测)

### 4. 训练配置
- **损失函数**: 均方误差(MSE)
- **优化器**: Adam优化器
- **学习率**: 0.001
- **批次大小**: 自适应调整(确保至少20个批次)

## 关键特性

### 1. 模块化设计
- `QuantumClassicalHybridModel`: 量子经典混合模型类
- `BeamDataProcessor`: 数据预处理类
- `PQCConfig`: 配置管理类

### 2. 数据类型安全
- 明确的数据类型转换(float32)
- 批次数据类型一致性检查

### 3. 可扩展性
- 配置驱动的参数管理
- 灵活的模型结构调整
- 标准化的输出格式

## 性能指标跟踪

训练过程中会监控以下指标：
- 训练损失(MSE)
- 验证损失(MSE)
- 测试集MSE、RMSE、R²

## 使用方法

```bash
# 运行完整训练
python pqc_ang_v1.py

# 验证配置修正
python test_pqc_corrected.py
```

## 注意事项

1. **内存管理**: 大数据集(336,000样本)需要足够的内存
2. **计算资源**: 量子电路计算较为耗时
3. **批次调整**: 根据可用内存自动调整批次大小
4. **早停机制**: 验证损失上升时自动停止训练

## 后续优化方向

1. **量子电路优化**: 尝试不同的量子门组合
2. **经典网络增强**: 更复杂的MLP结构

4. **分布式训练**: 支持多GPU并行计算
