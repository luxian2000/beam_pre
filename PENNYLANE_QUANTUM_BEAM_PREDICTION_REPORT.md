# Pennylane量子算法在波束RSRP预测中的应用分析

## 🎯 项目概述

本项目探索了使用Pennylane量子机器学习框架解决波束RSRP值预测问题的可行性。波束预测问题是典型的**多输出回归 + 缺失值插值**问题，具有高维输出(256维)和强相关性的特点。

## 🔬 技术方案

### 核心量子算法
1. **变分量子电路(VQC)** - 用于特征提取和相关性建模
2. **量子特征映射** - 将经典数据编码到量子态
3. **量子概率测量** - 提取量子特征用于经典机器学习

### 实现架构
```
输入数据 → 量子编码 → 变分电路 → 量子测量 → 经典回归器 → 预测输出
```

## 📊 实验结果分析

### 性能对比
| 方法 | MSE | R² Score | 相对于经典基线 |
|------|-----|----------|----------------|
| Classical Baseline | 0.555 | 0.443 | 基准(100%) |
| Quantum Random Forest | 1.164 | -0.169 | -109.8% |
| Quantum SVM | 1.032 | -0.036 | -86.1% |

### 关键发现

1. **当前阶段量子算法表现不佳**
   - 量子增强方法的MSE比经典基线高约86-110%
   - R²分数为负值，表明预测效果较差

2. **潜在原因分析**
   - 量子比特数量限制(仅4-8个量子比特)
   - 量子噪声和退相干影响
   - 数据编码效率有待优化
   - 变分优化收敛困难

## ⚡ 量子优势潜力

### 理论优势
1. **指数级加速**: 量子核函数计算
2. **高维数据处理**: 天然适合256维输出空间
3. **相关性建模**: 量子纠缠可捕获复杂波束关系
4. **并行处理**: 量子叠加实现同时计算

### 适用场景
- 大规模波束系统(>1000个波束)
- 实时在线预测需求
- 高维特征空间建模
- 复杂相关性分析
